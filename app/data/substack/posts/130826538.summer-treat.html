Our Weekly Bookmarks Bar

This week’s Bookmarks Bar is a special edition: the following links are some of our favorite free online learning resources for jump-starting folks who want to break into data science this summer.

Python Learning Track:

  • [Beginner Level: Think Python 2e] by Allen Downey. Also check out the rest of the Think series of books.

  • [Get Interactive: Python Tutor] by Philip Guo.

  • [Level Up: Python for Data Analysis 3E] by Wes McKinney.

  • [Get Interactive: Pandas Tutor] by Sam Lau and Philip Guo.

  • [Galaxy Brain: What are Embeddings?] by Vicki Boykis.

R Learning Track

  • [Beginner Level: R for Data Science 2e] by Hadley Wickham, Mine Çetinkaya-Rundel, and Garrett Grolemund.

  • [Beginner Level: R Programming for Data Science] by Roger Peng.

  • [Get Interactive: Tidy Data Tutor] by Sean Kross and Philip Guo.

  • [Level Up: An Introduction to Statistical Learning] by Gareth James, Daniela Witten, Trevor Hastie, and Rob Tibshirani.

  • [Galaxy Brain: Mastering Shiny] by by Hadley Wickham.
