Welcome to the sixteenth ever issue of Monday Morning Data Science from the Fred Hutch Data Science Laboratory. Fresh from the Lab

  • [Event: Data House Calls] Wednesday 2/22 at 10am we’ll have our weekly Data House Calls drop in hour on Teams. Drop by to talk about needed support for data challenges, coding challenges, computing questions, data management, etc.  If the DaSL team can’t help you, they will be able to ping several other folks with different skill sets (including SciComp, the DaSL training group, the Hutch Data Core, etc) and get the right connections for you.

  • [After Hours Event: Seattle R User Group] Wednesday 2/22 at 6pm the Seattle R User Group is meeting in the Arnold building M1-A303 where Melissa Gaughan, Brian Van Abbema, and Claire Evans from King County Metro will be presenting. This is a public event so please RSVP using the link above.
